# SubjectiveTwitterOnDemandDataSource

Subjective datasource implementation for SubjectiveTwitterOnDemandDataSource.

## Usage

```python
from subjective_datasources.SubjectiveTwitterOnDemandDataSource import SubjectiveTwitterOnDemandDataSource

source = SubjectiveTwitterOnDemandDataSource(params={})
source.fetch()
```

## Parameters

Use the params dictionary when constructing the datasource to provide connection and runtime values.
Refer to get_connection_data() for required fields.
